## Getting Started

### Prerequisites

- Flutter SDK (>=3.2.3)
- Dart SDK (>=3.2.3)
- Android Studio / VS Code
- Firebase account (for backend services)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Crayons001/RAISE.git
```

2. Navigate to the project directory:
```bash
cd RAISE/police_app
```

3. Install dependencies:
```bash
flutter pub get
```

4. Run the app:
```bash
flutter run
```
