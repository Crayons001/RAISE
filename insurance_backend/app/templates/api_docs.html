RAISE Insurance API Documentation

Authentication Endpoints

POST /api/auth/login

Login to get access and refresh tokens

POST /api/auth/refresh

Refresh access token using refresh token

POST /api/auth/logout

Logout and invalidate tokens

GET /api/auth/me

Get current user information

Accident Endpoints

GET /api/accidents/

Get all accidents (admin, insurance_agent, police)

GET /api/accidents/{id}

Get specific accident details

POST /api/accidents/

Create new accident report (police only)

PUT /api/accidents/{id}

Update accident report (admin, police)

Insurance Claim Endpoints

GET /api/insurance/claims

Get all insurance claims (admin, insurance_agent)

GET /api/insurance/claims/{id}

Get specific claim details

POST /api/insurance/claims

Create new insurance claim (insurance_agent)

PUT /api/insurance/claims/{id}

Update claim details (admin, insurance_agent)

PATCH /api/insurance/claims/{id}/status

Update claim status (admin, insurance_agent)
